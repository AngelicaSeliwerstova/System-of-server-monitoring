# System-of-server-monitoring

Программа умеет :
- Хранить события в виде объектов, а затем добавлять их в Monitor;
- Находить число различных levels, таких как WARNING, ERROR, INFO;
- Определять общий статус сервера, в зависимости от числа помех;
- Находить модуль(source), вызывающий больше всего проблем;
- Находить последний лог с ошибкой;
- Фильтровать события по уровню;
- Фильтровать события по модулю;
- Показывать отчет через консольное окно.
